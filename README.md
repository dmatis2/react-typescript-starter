  ## Description 
  
  This project is simple React template with Typescript, ESLint, Prettier & Husky running on Parcel bundler. It has even SCSS support!

  ## Usage 
  
  * download the project
  * run `npm i` or `yarn`
  * start working on your React components in `src/`
